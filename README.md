# 📝 DescriptoAI - Advanced Shopify Description Generator

DescriptoAI is a premium, aesthetic web application built with **Streamlit** and powered by **Google Gemini API** that helps e-commerce stores generate scroll-stopping, high-converting product descriptions instantly.

## ✨ Features
- **Lavender & Velvet Theme:** Sleek, modern, and high-end SaaS UI/UX with beautiful 3D particle elements.
- **Advanced Target Audience & Tone Controls:** Tailor descriptions perfectly for Gen Z, Millennials, or professional buyers.
- **SEO & Competitor Insights:** Input target keywords and reference competitor URLs for optimized copy.
- **Smart Formatting:** Choose between bullet points, storytelling, or standard paragraph layouts.

## 🛠️ Tech Stack
- Python
- Streamlit (with Custom CSS)
- Google Generative AI (Gemini API)
